About Us

  • Our system aims to increase driver safety with least cost and maximum functionality by implementing all of hardware interfaces to minimize the code footprint and by using the best out of our hardware to increase performance.

  • The system can be embedded in a real car system and communicate with it

  • It aims to be user friendly by through android application and a website to monitor user data and configure the system’s features.

Features

Accident Detection System

Using gyroscope and accelerometer accident is detected immediately ,then GPS determines the location of the car and send its location to MCU to send SMS to the emergency number.

Sleep Detection System

Using camera to determine whether the driver is drowsy through a system based on artifial intelligence.

AR Navigation

User sends the destination location via the app to the cloud, The cloud sends the directions and the distance to be displayed.

User Interface

User Interface through which you can monitor your data and to configure the system’s features.

System Integration

The system can be easily embedded in your car system and communicate with it.

OTA Update

An Over-The-Air update is the wireless delivery of new software to our system with a click of a button.
